Basics

  1. CSS tells the browser how to display your webpage.
    You can use CSS to set the color, font, size, and other aspects of HTML elements.
  2. You have three methods for applying CSS.
    1.You can directly add style within the HTML code.
    2.You can add style within the head section of the HTML.
    3.You're best to put all the styles in a separate file and link to it.
    Created a separate style.css file and switched the editor view to that file.
  3. You can add style to an element by specifying it in the style element and setting a property.
    For example, you can center your h1 element by setting its text-align property to the value center.
  4. You can add the same group of styles to many elements by creating a list of selectors.
    Each selector is separated with commas.
  5. You need to link the styles.css file in the HTML, so the styles will be applied.
    Nest a self-closing link element in the head element.
    Give it a rel attribute value stylesheet and an href attribute value of style.css.
  6. The 'rel' attribute specifies the relationship of the linked resource, and in this case, 'rel="stylesheet"' indicates that the linked resource is a style sheet file.
    This informs the browser that the file is a style sheet and needs to be applied to the document.
    However, the 'rel' attribute is not limited to just 'stylesheet'. It can be used to define different types of relationships. For example:
    • rel="stylesheet"：The linked resource is a style sheet file.
    • rel="icon"：Specifies the website's icon.
    • rel="preconnect"：Specifies pre-connecting resources.
    • rel="alternate"：Specifies alternate versions of resources, such as alternate language versions or documents.
  7. You can use the id selector to target a specific element with an id attribute.
    An id selector is defined by placing the hash symbol # directly in front of the element's id value.
  8. A class selector is defined by a name with a dot directly in front of it.
  9. Comments in CSS look like this：/* comment here */

  • Position:absolute：絕對位置，從版面(父元素)的左上角開始計算

The display property

用來定義 HTML 元素的顯示方式。
  1. block：block：元素呈現為區塊級元素，它會新起一行並且盡可能撐滿其父元素的寬度，可以設定寬度、高度、外邊距和內邊距。
    2. When two block elements are placed next to each other, they stack on top of each other like actual blocks. The new block is now occupying the entire width of the marker, pushing the old block down to the next line.
  2. inline：元素呈現為內聯元素，它不會產生換行，只佔據它實際內容的大小，不可設定寬度和高度，水平方向上的外邊距和內邊距會起作用，垂直方向上的外邊距和 內邊距則不會。
  3. inline-block：元素呈現為內聯塊元素，它不會新起一行，但可以設定寬度、高度、外邊距和內邊距。
  4. flex：元素將以彈性盒模型顯示，可用於建立靈活的佈局。
  5. grid：元素將以網格佈局顯示，用於建立複雜的二維佈局。
  6. none：元素將不會被顯示，完全不佔據頁面的空間。
  7. table：元素將呈現為表格元素，可透過 table-row、table-cell 等屬性來控製表格佈局。
  8. inline-table：元素呈現為內嵌表格元素，類似 table，但不會新起一行。
  9. list-item：元素呈現為清單項，通常與 list-style 屬性一起使用，用於清單樣式的設定。
  10. inherit：繼承父元素的 display 屬性。

Flexbox Layout

Flexbox is a powerful layout mode in CSS, mainly used for laying out and aligning elements in one dimension. Compared to the traditional layout methods based on float or position, Flexbox provides a more concise and flexible way to align and distribute elements. Flexbox includes two core concepts：flex container and flex items.

Flex Container

By setting display：flex (allowing child elements to produce various arrangement methods, note：at this time the child elements inside will become horizontally arranged) or display：inline-flex, an element is defined as a flex container. It provides layout control over two directions：the main axis and the cross axis.
The main properties include:
  1. flex-direction：sets the direction of the main axis.
    • row (default)：Horizontal axis with flex items from left to right.
    • row-reverse：Horizontal reversed with flex items from right to left.
    • column：Vertical axis with flex items arranged from top to bottom.
    • column-reverse：Vertical reversed axis with flex items arranged from bottom to top.
  2. justify-content：Sets the alignment of flex items along the main axis. (設置主軸上項目的對齊方式)
    • flex-start：Flex items are packed toward the start of the main axis.
    • flex-end：Flex items are packed toward the end of the main axis.
    • center：Flex items are centered along the main axis.
    • space-between：Flex items are evenly distributed with the first item at the start and the last item at the end of the main axis. (伸展到父元素的兩側)
    • space-evenly：Flex items are evenly distributed with equal space around them. (均分子元素以外的空隙)
  3. align-items：設置交叉軸上單行元素的對齊方式
    • stretch：這是默認值。如果 flex 項目沒有設置高度或設為 auto，將沿交叉軸方向被拉伸以佔據整個 flex 容器的高度。如果設置了固定的高度，則保持其高度不變。
    • flex-start：flex 項目沿著交叉軸的起始位置對齊。在主軸為水平且交叉軸為垂直方向時，就是頂部對齊。
    • flex-end：flex 項目沿著交叉軸的終止位置對齊。在主軸為水平且交叉軸為垂直方向時，就是底部對齊。
    • center：flex 項目在交叉軸上居中對齊。
    • baseline：根據 flex 項目中基線不同的部分對齊。適用於交叉軸為水平方向時，讓項目的基線對齊。
  4. flex-wrap：Sets whether flex items can wrap. (設置項目是否可換行)
    • nowrap (default)：Flex items will not wrap (single line).
    • wrap：Flex items will wrap onto multiple lines.
    • wrap-reverse：Flex items will wrap onto multiple lines in reverse order.
  5. align-content：Sets the alignment of flex lines when there is extra space on the cross axis. (設置多行交叉軸上的對齊方式)
    • stretch：Stretch flex lines to take up the remaining space.
    • flex-start：Flex lines are aligned at the start of the cross axis.
    • flex-end：Flex lines are aligned at the end of the cross axis.
    • center：Flex lines are centered along the cross axis.

Flex Items

Flex container 的直接子元素就是 flex items。它們可以通過以下屬性進行控制:
  1. order：定義項目的排列順序
  2. flex-grow：定義項目的放大比例
  3. flex-shrink：定義項目的縮小比例
  4. flex-basis：定義項目的初始長度
  5. align-self：允許單個項目有與其他項目不同的對齊方式
  6. object-fit(cover)：用於控制如何調整替換元素的內容大小以適應其容器。

The overflow property

The overflow property in CSS is used to control what happens to content that is too big to fit in its containing box.
  1. visible (default)：The content is not clipped and is rendered outside the element's box. (內容不會被修剪，會全部顯示在元素框之外)
  2. hidden：The content is clipped, and any excess is not shown. (內容會被修剪，多餘的部分隱藏不顯示)
  3. scroll：Scrollbars are added to the element to show the overflowing content, regardless of whether the content is larger than the box or not. (無論內容是否超出，都會顯示滾動條，以便查看多餘的內容)
  4. auto：Scrollbars are added only when necessary, i.e., when the content overflows the box. (只有在內容超出元素框時才顯示滾動條)
  5. overlay：Similar to auto, but the scrollbars are drawn on top of the content, rather than taking up space within the element's box. (行為類似 auto，但滾動條繪製在元素內容之上，而不是佔用內容空間)

The transform property

transform 是CSS中一個非常強大的屬性，它允許我們對元素進行2D或3D的幾何變換，包括旋轉(rotate)、縮放(scale)、傾斜(skew)、位移(translate)等操作。 這些函數可以單獨使用，也可以通過空格組合使用實現復合變換效果。 除了2D變換外，transform 還支持3D變換，例如rotateX() rotateY() translateZ()等函數。 transform 屬性的強大之處在於可以實現多種幾何變換效果，並且不會影響到元素在文檔流中的位置和大小，非常適合製作動畫效果。它也可以與transition和animation配合使用，實現平滑的動態效果。
  1. translate(x, y):平移變換，根據給定的x和y值將元素在平面上沿着x軸和y軸方向移動指定的距離。
    transform：translate(50px, 100px); /* 向右移動50px，向下移動100px */
  2. rotate(angle):旋轉變換，將元素繞自身中心點旋轉給定角度，正值為順時針方向。
    transform：rotate(45deg); /* 順時針旋轉45度 */
  3. scale(x, y):縮放變換，根據給定的x和y比例值縮放元素的尺寸。
    transform：scale(1.5, 0.8); /* 水平方向放大1.5倍，垂直方向縮小0.8倍 */
  4. skew(x-angle, y-angle):傾斜變換，沿X軸和Y軸方向傾斜給定的角度值。
    transform：skew(30deg, 20deg); /* 沿X軸方向傾斜30度，沿Y軸方向傾斜20度 */
  5. matrix():使用矩陣直接描述變換，更加通用和強大。

The box-sizing property

用於控制瀏覽器渲染元素的寬高計算方式。它有以下三個可能的值:

  1. content-box：這是瀏覽器默認的行為方式。元素的寬高僅包括 content 區域，border 和 padding 都不被計算在內。例如，如果一個元素設置了 width：200px; padding：20px;
    那麼實際渲染出來的寬度將是 200px + 20px*2 = 240px。這種情況下使得手動計算元素的實際佔用空間變得很麻煩。
  2. border-box：這種模式下，元素設置的 width 和 height 值已經包括了 border 和 padding，實際渲染出的寬高就等於設置的值。
    比如設置了 width：200px; padding：20px; border：2px solid black;，渲染出來的寬度還是 200px，內容區域寬度是 200 - 202 - 22 = 156px。
    這種模式使得設置元素尺寸變得簡單直觀。
  3. padding-box（實驗性，支持度較差）：一種過渡的盒模型，元素的寬高包含了 padding，但不包含 border。

大多數開發者傾向於使用 border-box 模式，因為這樣計算元素尺寸更簡單直覺。一些流行的 CSS 框架如 Bootstrap 也設置了全局的 box-sizing：border-box。

There are two main color models:

The additive RGB (red, green, blue) model used in electronic devices, and the subtractive CMYK (cyan, magenta, yellow, black) model used in print.
There are two ways to set the color value：
1. color name.
2. rgb(red, green, blue)：red/green/blue are 0~255 or 0~100%;

  • Secondary colors are the colors that result from combining primary colors.

Complementary colors

A color wheel is a circle where similar colors, or hues, are located near each other, and different ones are further apart. Two colors that are opposite each other on the color wheel are called complementary colors. If two complementary colors are combined, they produce gray. When placed side by side, these colors create a strong visual contrast and appear brighter. This contrast can be distracting if it is overused on a website and can make text hard to read when placed on a complementary-colored background. It is better practice to select one color as the dominant color and use its complementary color as an accent to draw attention to specific content on the page.

Hexadecimal / hex color values

A very common way to apply color to an element with CSS is using hexadecimal or hex values. Hex color values start with a # character and consist of six characters ranging from 0-9 and A-F. The first pair of characters represents red, the second pair represents green, and the third pair represents blue. With hexadecimal colors, 00 represents 0% of that color, and FF represents 100%.
  1. #FF0000：red
  2. #00FF00：green
  3. #0000FF：blue

The HSL color model：hsl(hue, saturation, lightness)

The HSL color model, which stands for hue, saturation, and lightness, is an alternative method for representing colors. For example, hsl(hue, saturation, lightness). The CSS hsl function accepts three values：a number from 0 to 360 for hue, a percentage from 0 to 100 for saturation, and a percentage from 0 to 100 for lightness.
If you imagine a color wheel, the hue red is at 0 degrees, green is at 120 degrees, and blue is at 240 degrees.
Saturation is the intensity of a color ranging from 0% (gray) to 100% (pure color). You must add the percent sign (%) to the saturation and lightness values.
Lightness is the brightness level of a color, ranging from 0% (complete black) to 100% (complete white), with 50% representing neutrality.

Gradient：linear-gradient(gradientDirection, color1, color2, ...)

The CSS linear-gradient function allows you to control the direction of the transition along a line and determine which colors are used. One thing to remember is that the linear-gradient function actually creates an image element and is usually paired with the background property, which can accept an image as a value.
Basic syntax：linear-gradient(gradientDirection, color1, color2, ...);
For example：background：linear-gradient(90deg, rgb(255,0,0), rgb(0,255,0));
Gradient direction refers to the direction of the line used for the transition. Color1 and color2 are arguments representing the colors that will be used in the transition itself. These can be any type of color, including color keywords, hex, rgb, or hsl. Color stops allow you to fine-tune where colors are placed along the gradient line. They are a length unit like pixels (px) or percentages (%) that follow a color in the linear-gradient function.
For example：background：linear-gradient(90deg, rgb(255, 0, 0) 50%, rgb(0, 255, 0), rgb(0, 0, 255));
Even if you don't set a color stop point, the linear gradient function automatically calculates these values for you and presets to place the color evenly along the gradient line. If no gradient direction argument is provided to the linear-gradient function, it arranges colors from top to bottom, or along a 180-degree line, by default.

Opacity

  • RGB：rgba(redValue, greenValue, blueValue, alphaValue)
  • HEX：#Red Green Blue Alpha
  • HSL：hsl(hue, saturation, lightness, alphaValue)
Opacity describes how opaque, or non-transparent, something is. For example, a solid wall is opaque, and no light can pass through. But a drinking glass is much more transparent, and you can see through it to the other side. With the CSS opacity property, you can control the level of opacity or transparency of an element. With the value 0, or 0%, the element will be completely transparent, and at 1.0, or 100%, the element will be completely opaque, as it is by default. Another way to adjust the opacity of an element is by using the alpha channel. Similar to the opacity property, the alpha channel controls the level of transparency or opacity of a color. The 'rgba' function works similarly to the 'rgb' function, but it requires an additional number ranging from 0 to 1.0 for the alpha channel.
For example, rgba(redValue, greenValue, blueValue, alphaValue);

Shadow：box-shadow：offsetX offsetY blur_radius spread_radius color;

The box-shadow property allows you to apply one or more shadows around an element.
Here's how the offsetX and offsetY values work:
  • Both offsetX and offsetY accept numerical values in pixels (px) and other CSS units.
  • A positive offsetX value moves the shadow to the right, and a negative value moves it to the left.
  • A positive offsetY value moves the shadow down, and a negative value moves it up.
  • If you want a value of zero (0) for either or both offsetX and offsetY, you don't need to add a unit. Every browser understands that zero means no change.
The height and width of the shadow are determined by the height and width of the element to which it is applied. You can also use an optional spreadRadius value to extend the reach of the shadow. Notice that the edges of the shadow are sharp. This is because there is an optional blurRadius value for the box-shadow property.
For example, box-shadow：offsetX offsetY blurRadius color;
If a blurRadius value isn't included, it defaults to 0 and produces sharp edges. The higher the value of blurRadius, the greater the blurring effect will be. If you want to extend the shadow further, you can achieve this by using the optional spreadRadius value.
Use the "inset" keyword of box-shadow to create an inner shadow effect. By adjusting the blur radius and extension radius, you can blur the edges of the inner shadow, thereby achieving the effect of blurring the edges of the background.
For example, box-shadow：0 0 20px 20px #fff inset

The filter property：用於對元素應用圖形效果，比如模糊、銳利化、變色等。它提供了一系列功能函數，可以單獨使用或組合使用。

  1. filter：blur(4px)：應用高斯模糊效果，值越大越模糊。
  2. filter：brightness(150%)：調整圖像的亮度。
  3. filter：contrast(200%)：調整圖像的對比度。
  4. filter：grayscale(100%)：將圖像轉換為灰階。
