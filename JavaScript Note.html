Highlight plugin

1.You can download the latest version of Highlight.js from the official website (https://highlightjs.org/).
Choose the programming language support you need, or you can also download the full version.
2.Unzip the downloaded file and include it in your project.
You can place the files in a suitable location within your project directory, such as a folder named 'highlight'.
3.Add the Highlight.js stylesheet to your webpage.
Include the following code in the <head> tag to link the Highlight.js stylesheet file:
(I have selected the github-dark.css for my styles.) 

    <link rel="stylesheet" href="./highlight/styles/github-dark.css">
4.Include the following code at the bottom of the <body> tag (recommended) or within the <head> tag to import the Highlight.js script:

    <script src="./highlight/highlight.js"></script> 
    <script>hljs.highlightAll();</script>
5.After your webpage has finished loading, you typically initialize Highlight.js within a <script> tag.
You can add the following code at the bottom of the <body> tag:

    <script>
        document.querySelectorAll('pre code').forEach((block) => {
            if (block.dataset.highlighted) {
                delete block.dataset.highlighted;
            }
            hljs.highlightElement(block);
            block.dataset.highlighted = 'yes';
        });
    </script>
6.Now, you can use the <pre><code> tags to include code in your webpage.
Highlight.js will automatically syntax highlight these code blocks.

    <div class="program">
        <pre><code class="language-html">

        </code></pre>
    </div>
